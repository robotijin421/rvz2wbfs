Conversor de RVZ a WBFS

Necesita que estén instalados Dolphin https://dolphin-emu.org y Wit https://wit.wiimm.de/download.html 

Puedes arrastar archivos RVZ o Zip y dejarlos ya en una carpeta de salida. Si el archivo es mayor de 4GB lo divide automáticamente y le añade el GameID al nombre del archivo.

El path tanto de dolphin y de wit tiene que incluir sus respectivos ejecutables, dolphin-tool.exe y wit.exe, es decir, es la ruta con el archivo.
